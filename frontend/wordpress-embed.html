DentistEnvy WordPress Integration

Embed the SEO comparison tool on newpatientsinc.com

Quick Start

Add this code to any WordPress page or post where you want the widget to appear:

<!-- DentistEnvy SEO Comparison Tool --> <div data-dentistenvy></div> <script> window.DENTISTENVY_API_URL = 'https://your-api-domain.com'; </script> <script src="https://your-api-domain.com/embed.js"></script>
Note: Replace https://your-api-domain.com with your actual API server URL. For local testing, use http://localhost:5001.

WordPress Installation Steps

1 Create a New Page

In WordPress admin, go to Pages → Add New

2 Switch to HTML/Code Editor

Click the three dots menu (⋮) in the top right and select "Code editor" or use a Custom HTML block

3 Paste the Embed Code

Copy and paste the embed code above into the editor

4 Publish the Page

Click Publish to make the page live

Using with Elementor

If you're using Elementor page builder:

  • Drag an "HTML" widget onto your page
  • Paste the embed code into the HTML widget
  • Save and preview

Using with WPBakery

If you're using WPBakery (Visual Composer):

  • Add a "Raw HTML" element
  • Paste the embed code
  • Save changes

Shortcode Option

To create a reusable shortcode, add this to your theme's functions.php:

function dentistenvy_shortcode() { return ' <div data-dentistenvy></div> <script> window.DENTISTENVY_API_URL = "https://your-api-domain.com"; </script> <script src="https://your-api-domain.com/embed.js"></script> '; } add_shortcode('dentistenvy', 'dentistenvy_shortcode');

Then use [dentistenvy] anywhere in your content.

For Local Testing

To test locally before deploying:

<!-- Local Testing --> <div data-dentistenvy></div> <script> window.DENTISTENVY_API_URL = 'http://localhost:5001'; </script> <script src="http://localhost:5001/embed.js"></script>
Test Page: Visit http://localhost:5001 to see the standalone version, or view these instructions.

Production Deployment

When deploying to production, you'll need to:

  • Host the Flask API on a server (Railway, Render, AWS, etc.)
  • Set up HTTPS for secure connections
  • Update the API URL in the embed code
  • Configure CORS to allow requests from newpatientsinc.com

Troubleshooting

  • Widget not appearing? Check browser console for JavaScript errors
  • CORS errors? Make sure the API server allows requests from your WordPress domain
  • Styles look wrong? The widget uses scoped styles with "de-" prefix to avoid conflicts
